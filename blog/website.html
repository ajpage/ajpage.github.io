Creating a Website / reflection

February 26, 2016

Alyssa Page

Creating a website has been a fun and at times a frustrating experience. It's fantastic to be able to build something from start to (somewhat) finish. With this comes a lot of learning, and the more I have learned the more I want to change. I can see how easy it could be to never be satisfied with the final product and always want to be tinkering with something to make it better. I guess thats one of the perks of having my own site though. I can play with it as much as I'd like to make it better or change anything I don't like. I'm excited to use this space to continue learning and growing and making improvements.
